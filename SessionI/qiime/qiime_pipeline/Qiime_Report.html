Welcome to QiimeReport!

by D. Giovannelli @ Deep-Sea Microbiology Lab - Rutgers University

version 0.2

QiimeReport is an automatic html report that easily and automatically shows the results from the Qiime pipeline for the analysis of 16S rRNA pyrotag libraries.

  • Summary of the library analysis results. Contains total number of sequences, sequences for samples and total identifies OTUs at 97% cut-off.
  • Qiime and reference database version. Contains the print out of qiime config to visualize the version of the software used and the analysis date.
  • Bar plot summary of the results. Contains the results of the diversity analysis at the Phylum, Class, Order, Family and Genus.
  • Rarefaction plots for the libraries. Contains the curve for the observed number of species (OTUs), Chao1 and Shannon diversity indexes. Select in the menu the metric to be visualized and in the category "SampleID".
  • Bar plot summary of the core microbiome (100 % samples).Contains the results of the diversity analysis at the Phylum, Class, Order, Family and Genus for the core microbiome (aka OTUs present in 100 % of samples).
  • Heatmap of the results. They can be visualized as OTUs number or taxonomic consensus level.
  • Phinch visualization. Opens the free phinch visualization tool. Load the custom .biom file to visualize your data.
  • Unweighted e Weighted PCoA. Will open the interactive page of Emperor to visualize and modify the PCoA of the library.
  • Representative fasta sequences of each OTU. It can be used to run further analyses (e.g. blastn or EzTaxon) on specific OTUs identified in the previous heatmap plot as abundante and/or of interest. it can be used to confirm Qiime taxonomic assignment confidence and % similarity of specific groups to reference sequences.
  • Diversity tree of the samples combined in Newick format obtained with the Qiime fasttree algoritm. Right click on the link (or apple+click) and save the file. Can be opened with FigTree, geneious or other tree drawing programs.
